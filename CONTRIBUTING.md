# Contributing to mineball

Thank you for your interest in mineball!

## This is a Meme Experiment

mineball is primarily a meme experiment and learning project. While we appreciate community interest, please understand:

- This is not a professional product
- We make no promises about maintenance
- Use at your own risk

## Reporting Issues

If you find a bug or security vulnerability:

1. **Security Issues**: Please email (add contact) privately
2. **Bugs**: Open a GitHub issue with:
   - Clear description
   - Steps to reproduce
   - Expected vs actual behavior
   - Your environment (OS, Node version, etc.)

## Suggesting Features

Feature requests are welcome, but remember this is an experiment. Open an issue with:
- Clear description of the feature
- Why it would be useful
- How it fits the mineball narrative

## Code Contributions

We're not actively seeking code contributions, but if you want to submit a PR:

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests if applicable
5. Ensure all tests pass
6. Submit a PR with clear description

### Code Style

- Follow existing code style
- Add comments for complex logic
- Keep functions small and focused
- Write tests for new functionality

### Commit Messages

Use clear, descriptive commit messages:
```
feat: add new feature
fix: resolve bug
docs: update documentation
test: add tests
refactor: restructure code
```

## Questions

For questions about the project:
- Read the [documentation](docs/)
- Check existing issues
- Open a new issue if needed

## Code of Conduct

Be respectful. Be professional. Be kind.

---

**Remember**: This is a meme experiment. Don't take it too seriously. Have fun! ⛏️
